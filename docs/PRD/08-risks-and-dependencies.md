# 8. 风险与依赖

## 8.1 技术风险

| 风险 | 影响程度 | 缓解措施 |
|------|----------|----------|
| JWT Token 泄露 | 高 | 支持 Token 黑名单、短有效期 + 刷新 |
| 数据库单点故障 | 高 | 主从复制、读写分离 |
| 缓存穿透 | 中 | 布隆过滤器、空值缓存 |
| 分布式事务 | 中 | 最终一致性、消息队列 |

## 8.2 依赖项

| 依赖 | 用途 | 备选方案 |
|------|------|----------|
| MySQL | 主数据库 | PostgreSQL |
| Redis | 缓存、会话存储 | 无（强依赖） |
| Kafka | 异步日志、消息队列 | RabbitMQ |
